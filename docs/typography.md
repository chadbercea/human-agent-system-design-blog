# Typography

**Superseded.** The old stack (Poppins / IBM Plex Mono / Fraunces, brutalist editorial) described **the canceled direction**.

Current typography is defined in the **[HAS Design Blog agent briefing](https://linear.app/iliketobuild/document/claudemd-has-design-blog-agent-briefing-5f16dbea3f75)** and implemented per **ILI-561**: serif for prose, monospace for metadata/code, self-hosted WOFF2 in `public/fonts/`, tokens in `src/styles/typography.css` and `fonts.css` — no Google Fonts CDN, two fonts maximum.
